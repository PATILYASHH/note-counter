Frequently Asked Questions

How do I report a bug?

The best way to report bugs is through our GitHub Issues page. This allows us to track the issue, ask follow-up questions, and notify you when it's fixed. Please include as much detail as possible about what you were doing when the bug occurred.

Can you add support for my currency?

Yes! We're always looking to add support for more currencies. Please create a feature request on GitHub with details about the currency and its denominations. Include the currency code, symbol, and all available note and coin values.

Is Note Counter really free?

Absolutely! Note Counter is completely free to use with no hidden fees, subscriptions, or premium features. We believe in providing valuable tools accessible to everyone. The application is also open source, so you can verify this yourself.

Can I use Note Counter for commercial purposes?

Yes! Note Counter is released under the MIT License, which allows both personal and commercial use. Many businesses already use it for daily cash counting, till reconciliation, and financial reporting.

How can I contribute to the project?

We welcome contributions! You can contribute by reporting bugs, suggesting features, improving documentation, or submitting code changes. Visit our GitHub repository to get started with contributing guidelines.
