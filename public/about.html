Our Story

Note Counter was born from a simple observation: counting money accurately and efficiently shouldn't be complicated or expensive. Our founder, Yash Patil, noticed that many small businesses and individuals struggled with cash management due to lack of proper tools or expensive solutions.

March 2024

Initial concept and development began with a focus on creating a simple, effective money counting tool.

June 2024

First public release with basic counting features and support for major currencies.

September 2024

Added mobile optimization and text format display based on user feedback.

December 2024

Implemented PWA features, calculator integration, and history tracking.

January 2025

Open-sourced the project and launched comprehensive privacy protections.

Technology & Innovation

Note Counter is built using modern web technologies to ensure reliability, speed, and accessibility:

Technical Stack

  • Frontend: React 18, TypeScript, Tailwind CSS
  • Build System: Vite for fast development and optimized production builds
  • PWA: Progressive Web App capabilities for offline usage
  • Icons: Lucide React for consistent, beautiful iconography
  • Deployment: Netlify for reliable hosting and continuous deployment

Privacy & Security

We take privacy seriously. Note Counter operates on a "privacy by design" principle:

  • All calculations performed locally in your browser
  • No data sent to external servers
  • No tracking, analytics, or user profiling
  • Local storage only for user preferences
  • Open source code for full transparency
