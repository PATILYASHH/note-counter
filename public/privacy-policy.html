2. Information We Collect

2.1 Automatic Data Collection

When you visit Note Counter, we automatically collect certain information:

  • Location Data: Country and regional information via IP geolocation (collected once per device)
  • IP Address Hash: Privacy-friendly hashed identifier to prevent duplicate data submissions
  • Browser Information: User agent, browser type, and version for compatibility
  • Device Information: Device type, operating system
  • Timezone: User's timezone for service optimization
  • Usage Data: Pages visited, time spent, features used

2.2 Local Storage Data

The following data is stored locally on your device:

  • Currency counting sessions and denomination counts
  • Application preferences (currency selection, display settings)
  • History of counting sessions
  • Custom currency configurations
  • Privacy mode settings

Note: This data remains on your device and is not transmitted to our servers unless you explicitly use cloud sync features.
